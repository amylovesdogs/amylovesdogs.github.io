Project: County Commissioner Redistricting

Analysis aid for proposed district maps

This project focuses on the county commissioner redistricting happening now (2023) in El Paso County, CO. The browser application allows the user to choose a proposed district map and view population, demographics and political competitiveness for each district on that map.

Technologies used:

  • Client side: Javascript, Leaflet, D3, Bootstrap and Chart.js.
  • Server side: Python, SQLite3.
